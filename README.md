# Aram Zêrevan — Personal Portfolio & Engineering Showcase

Welcome to my personal portfolio repository! This is a production-grade, highly interactive portfolio website engineered using **Next.js** and **TypeScript** to showcase my full-stack web architectures, UI/UX designs, and software tools.

---

## 🚀 Tech Stack & Core Technologies

- **Framework:** Next.js (App Router) & React
- **Language:** TypeScript / JavaScript
- **Styling:** Advanced CSS (Custom Properties, Glassmorphism, Responsive Grid Systems)
- **Animations & 3D:** GSAP (GreenSock Engine) & Three.js for hardware-accelerated WebGL visuals
- **Hosting:** Deployed via Vercel with automated CI/CD

---

## ✨ Key Features

- **Fluid Animations:** High-performance UI component transitions and immersive 3D spatial tilts powered by GSAP.
- **Production-Ready Architecture:** Clean folder structure exploiting Next.js App Router optimization, dynamic loading, and server-side components.
- **Instrument-Inspired Design:** A clean, modern dark-mode aesthetic with high-contrast typography focused on raw digital utility.
- **Responsive Viewports:** Fully fluid design layouts tailored to look flawless from mobile environments to ultrawide desktop monitors.
